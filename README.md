# news_app
 News Application

Overview:
Nwes is a streamlined news app enabling users to easily browse and personalize news articles. Featuring Dio package integration for efficient API calls, the app offers a seamless experience with categorized news and Google integration for expanded details.

Key Learnings:

Dio Package for HTTP requests.
Endpoints implementation for news retrieval.
API integration for general news.
Robust error handling and bug fixing.
JSON data display for a user-friendly interface.
Scrollable design for improved navigation
